# System Monitoring Script

## Description
This Bash script monitors system resources including CPU usage, memory usage, and disk usage. It logs the output to a file with a timestamp.

## Files
- `system_monitor.sh`: The Bash script for system monitoring.
- `system_monitor.log`: Sample log output after running the script.
- `README.md`: Instructions for running the script and setting up the cron job.

## Running the Script
Make sure the script is executable:
```bash
chmod +x system_monitor.sh
